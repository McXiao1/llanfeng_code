# Product

## Register

product

## Users

Windows developers who install, configure, and switch Codex and Claude Code
providers. They use the application as a focused desktop utility and expect
configuration tasks to be quick, predictable, and reversible.

## Product Purpose

Llanfeng Code Assistant centralizes provider profiles, model metadata, and local
CLI configuration. Success means users can fetch, inspect, edit, activate, and
recover provider settings without manually editing configuration files.

## Brand Personality

Quiet, precise, and dependable. The interface should feel compact and polished,
with the tool receding behind the configuration workflow.

## Anti-references

Avoid marketing-style composition, decorative cards, gradients, emoji icons,
oversized typography, ornamental motion, and unfamiliar replacements for
standard form controls.

## Design Principles

- Keep repeated configuration work compact and easy to scan.
- Preserve familiar desktop form and dialog behavior.
- Separate user-facing labels from provider-facing identifiers.
- Make destructive or failing operations explicit and recoverable.
- Reuse the project's existing controls and visual vocabulary.

## Accessibility & Inclusion

Preserve native keyboard and focus behavior, provide text labels or tooltips for
icon controls, maintain readable contrast, and never rely on color alone to
communicate state. Keep motion optional and limited to state feedback.
